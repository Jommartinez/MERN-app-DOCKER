# SPA MERN WITH DOCKER

## BUILD
```bash
docker build -t react-app ./client/
docker build -t api-server ./server/
```
## RUN
```bash
docker-compose up
```
## STOP
```bash
docker-compose down
```
# URLs

#### Create product
    http://localhost:3000/products/create
### List products
    http://localhost:3000/products/list
